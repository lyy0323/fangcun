# 方寸 · 迭代计划

## v1.3 ✅ (2026-04-03)

- 导出图片：Canvas 渲染诗词卡片，5 套配色方案，2x 高清输出，带 logo 水印
- 图片排版：标题竖排自动断列、诗居中/词左对齐、标题色块装饰、按内容自适应高度
- Google Fonts 按需子集加载（Noto Serif SC），隔离字体不污染主界面
- 导入导出：批量导出/导入所有画板为 JSON 文件，按 ID 去重

## v1.2 ✅ (2026-04-02)

- 开放 API：新增 6 个 HTTP API 端点，供外部开发者和 AI Agent 调用
- API 安全：IP 限流、输入校验、安全响应头、API Key 认证（SHA256 哈希存储）
- API 文档：/docs 页面，自包含 HTML，含完整端点说明和示例
- CLI 工具：pip install 后可用 `fangcun` 命令行进行格律校验
- 数据库迁移：API Key 存储支持 SQLite（本地）/ Postgres（线上）双后端
- 前端免认证：同源请求自动跳过 API Key 检查

## v1.1 ✅ (2026-03-18)

- 词库扩容：数据源替换为 data2_with_metadata.json (45万首) + 私域数据，词首/词末/对语全面重建
- TOP_K 提升至 100，MIN_COUNT_PAIR 放宽至 {1:3, 2:2, 3:2, 4:2}
- 字典实时搜索：输入即查，300ms 防抖，移除搜索按钮
- 字典区高度统一：展开时固定 255px，各 tab 高度一致不再跳动
- 字典区收起/展开：抽拉控件 + 高度过渡动画
- 加载态优化：查询期间保留旧结果，opacity 半透明提示刷新中
- 设置面板：毛玻璃弹窗，含教程（带按钮图标）、导入导出（预留）、关于方寸

- 重字提醒：检测非叠词的重复用字，底部黄色警告提示
- 字典区与灵感板底部对齐优化
- 修复灵感板文本不能保存换行的问题
