# 方寸 · 迭代计划

## v1.6 ✅ (2026-04-08)

### 导入前人作品
- 新建画板页面新增「导入前人作品」入口，可搜索历代诗词库（86万首）并导入到画板
- 搜索支持关键词、字段筛选（标题/作者/内容）、朝代、体裁过滤
- 点击搜索结果查看完整内容及格律信息，仅允许导入已匹配格律规则的作品
- 导入后自动新建画板，填入标题与正文，即时触发格律校验
- 搜索与详情加载使用骨架屏动画

## v1.5 ✅ (2026-04-08)

### 典故检索
- 字典区新增「典故」标签页，输入单字/词语即可检索包含该字的典形词（数据源 ~13,000 条）
- 点击典形词弹出毛玻璃弹窗，展示释义、典源出处、同源典形词、古诗例句
- 弹窗内同源典形词可点击填入画布（画布点击跳转时锚定原字位置，手动搜索时从光标向后填充）
- 排序策略：精确匹配优先 → 匹配位置靠前 → 短词优先（2-4 字适合入诗）→ 同源越多越经典

### 词的换行修复
- 词（Ci）分行逻辑支持「叶」韵位置换行（此前仅在 SAME_CATEGORY 韵脚处换行，遗漏 RELATION 叶韵）
- 韵脚字区域同步显示叶韵位置的字（如西江月过片处）

### UI 统一与优化
- 所有下拉菜单统一为自定义样式（韵书切换、TopBar 导出菜单、元数据公历/农历切换），移除原生 select
- TopBar 右侧：复制文字 / 导出图片 / 上传南洋吟游 合并为统一导出菜单
- TopBar 设置按钮移至右侧区域
- 画板切换、元数据弹窗上沿与 TopBar 下沿齐平
- 元数据日期输入框：空值时按 Tab 填入当天日期占位符
- 导出图片：词（Ci）的日期左对齐，与正文对齐方式一致

### 图片导出排版优化
- 画布比例动态适配：根据内容量自动选择 3:4 → 9:16 → 9:18，超长内容动态加长
- 字号与画布高度解耦：字号仅由内容量决定，画布高度按需匹配最小标准比例
- 词（Ci）大字排版：每行不超过 15 字时使用与律诗相同的 44px 字号
- 可伸缩留白区域集中在标题与正文之间，正文及以下间距固定

### 待开发
- [ ] 署名功能：新作可自行署名，导入旧作绑定原作者署名（#5）
- [ ] 导出图片标题使用更古典的字体：需支持实时子集化下载（#2，assigned @lyy0323）
- [ ] 韵书支持周行新韵、中原音韵、先秦上古韵
- [ ] 词谱增加换头位置参数（section_starts），用于排版分阕
  - **需求**：词分单调/双调/三叠等多段，每个词牌体例需记录各阕的起始字符位置，以支持前端分阕排版
  - **方案**：在 `ci_rules.json` 每条记录中增加 `"section_starts": [N1, N2, ...]`（0-based，指第 2/3/… 阕在 tone_pattern 中的起始 index）；同步更新 `config_loader.py`（RuleSet 增加字段）、`app.py`（API 序列化）、`types.ts`（ClosestRule 接口）
  - **踩坑**：源数据 `ci_data/*.json` 的 `paragraphs` 数组并非按阕划分，部分 paragraph 是句内分段而非换头分阕，因此不能简单按 paragraph 累计字数得到换头位置。需要额外解析 `description` 字段（如"双调xx字，前段xx句…后段xx句…"）或人工标注，才能准确提取阕边界
